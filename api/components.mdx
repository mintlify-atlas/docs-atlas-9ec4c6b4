## AdminPanel

Admin control panel that appears for contract owners. Provides buttons to manage voting phases and withdraw funds.

### Import

```typescript
import { AdminPanel } from "@/components/AdminPanel";
```

### Props

This component has no props - it uses the `useMemberVote` hook internally.

### Behavior

- Only renders if connected wallet is the contract owner
- Displays as a fixed bottom panel with backdrop blur
- Shows different controls based on workflow phase

### Rendered Controls

<ParamField body="Open Polls" type="button">
  Visible during `REGISTERING` (0) or `RESETTING` (2) phases. Calls `startVote()` to begin voting.
</ParamField>

<ParamField body="Close & Tally" type="button">
  Visible during `VOTING` (1) phase. Calls `resetVotes()` to end voting and tally results.
</ParamField>

<ParamField body="Collect {amount} ETH" type="button">
  Visible when contract balance > 0. Calls `withdrawFunds()` to transfer ETH to owner.
</ParamField>

### Usage Example

```typescript src/app/page.tsx
import { AdminPanel } from "@/components/AdminPanel";
import { DashboardGrid } from "@/components/DashboardGrid";

export default function Home() {
  return (
    <main>
      <DashboardGrid />
      <AdminPanel /> {/* Automatically hidden for non-owners */}
    </main>
  );
}
```

## VotingCard

Displays a voting option with live vote count, percentage bar, and vote button.

### Import

```typescript
import { VotingCard } from "@/components/VotingCard";
```

### Props

<ParamField body="title" type="string" required>
  Display name for the voting option (e.g., "Crispy Apple")
</ParamField>

<ParamField body="optionId" type="number" required>
  Option identifier: `0` for Option A, `1` for Option B
</ParamField>

<ParamField body="votes" type="number" required>
  Current vote count for this option
</ParamField>

<ParamField body="totalVotes" type="number" required>
  Combined votes for both options (used to calculate percentage)
</ParamField>

<ParamField body="hasVoted" type="boolean" required>
  Whether the connected user has already voted in this election
</ParamField>

### TypeScript Interface

```typescript
interface VotingCardProps {
  title: string;
  optionId: number;
  votes: number;
  totalVotes: number;
  hasVoted: boolean;
}
```

### Usage Example

```typescript src/components/DashboardGrid.tsx
import { VotingCard } from "@/components/VotingCard";
import { useMemberVote } from "@/hooks/useMemberVote";

export const DashboardGrid = () => {
  const { optionAVotes, optionBVotes, hasVoted } = useMemberVote();
  const totalVotes = optionAVotes + optionBVotes;

  return (
    <div className="grid grid-cols-1 lg:grid-cols-2 gap-8">
      <VotingCard 
        title="Crispy Apple" 
        optionId={0} 
        votes={optionAVotes} 
        totalVotes={totalVotes} 
        hasVoted={hasVoted}
      />
      <VotingCard 
        title="Sweet Banana" 
        optionId={1} 
        votes={optionBVotes} 
        totalVotes={totalVotes} 
        hasVoted={hasVoted}
      />
    </div>
  );
};
```

## DashboardGrid

Main dashboard component that orchestrates the voting interface based on workflow phase.

### Import

```typescript
import { DashboardGrid } from "@/components/DashboardGrid";
```

### Props

This component has no props - it manages its own state via `useMemberVote`.

### Workflow Phases

<Tabs>
  <Tab title="REGISTERING (0)">
    Shows a standby screen with sparkle icon
    
    ```typescript
    {workflowStation === 0 && (
      <div className="flex flex-col items-center">
        <Sparkles className="w-16 h-16 animate-pulse" />
        <h2>Governance Standby</h2>
        <p>Voting will commence shortly.</p>
      </div>
    )}
    ```
  </Tab>
  
  <Tab title="VOTING (1)">
    Displays the active voting interface with both VotingCards
    
    ```typescript
    {workflowStation === 1 && (
      <div className="space-y-16">
        <h2>{election.question}</h2>
        <div className="grid grid-cols-1 lg:grid-cols-2 gap-8">
          <VotingCard title={election.optionA} optionId={0} ... />
          <VotingCard title={election.optionB} optionId={1} ... />
        </div>
      </div>
    )}
    ```
  </Tab>
  
  <Tab title="RESETTING (2)">
    Shows results screen with winner announcement
    
    ```typescript
    {workflowStation === 2 && (
      <div className="flex flex-col items-center">
        <Trophy className="w-20 h-20" />
        <h2>{winner}</h2>
        <p>Community Pool: {prizePool} ETH</p>
      </div>
    )}
    ```
    
    Winner determined by:
    ```typescript
    optionAVotes === optionBVotes ? "Split Decision" : 
    optionAVotes > optionBVotes ? election.optionA : election.optionB
    ```
  </Tab>
</Tabs>
