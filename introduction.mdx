    <p className="text-base text-gray-600 dark:text-gray-400">Get up and running with VoteLab in minutes</p>
  </div>

  <Steps>
    <Step title="Connect Your Wallet">
      Click the **Connect Wallet** button in the header to connect your Web3 wallet using RainbowKit. VoteLab supports all major wallet providers including MetaMask, WalletConnect, and Coinbase Wallet.
      
      ```typescript
      // Wallet connection is handled automatically via RainbowKit
      const { address, isConnected } = useAccount();
      ```
    </Step>

    <Step title="Review Election Status">
      Check the current workflow state to see if voting is open. The contract operates in three states:
      - **Registering**: Admin is preparing the election
      - **Voting**: Polls are open for participation
      - **Resetting**: Election closed, results locked

      ```typescript
      const { workflowStation } = useMemberVote();
      // 0 = Registering, 1 = Voting, 2 = Resetting
      ```
    </Step>

    <Step title="Cast Your Vote">
      When voting is open, select your preferred option and click **Cast Your Vote**. You'll need to pay the entry fee of 0.01 ETH along with gas fees.

      ```typescript
      const { castVote, isPending } = useMemberVote();
      
      // Vote for Option A (ID: 1) or Option B (ID: 2)
      castVote(1);
      ```

      <Note>
        Each wallet can only vote once per election. Your vote is recorded on-chain and cannot be changed.
      </Note>
    </Step>

    <Step title="View Results">
      After voting closes, the dashboard displays the winner and final vote counts. Entry fees form a prize pool that the admin can withdraw.

      ```typescript
      const { optionAVotes, optionBVotes, prizePool } = useMemberVote();
      
      console.log(`Option A: ${optionAVotes} votes`);
      console.log(`Option B: ${optionBVotes} votes`);
      console.log(`Prize Pool: ${prizePool} ETH`);
      ```
    </Step>
  </Steps>
